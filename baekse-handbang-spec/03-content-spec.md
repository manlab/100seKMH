# 03. 페이지별 콘텐츠 명세

> 이 문서는 **각 페이지의 섹션 구성, 카피라이팅 톤, 필요한 자산**을 정의합니다.
> 카피 본문은 모두 **샘플(예시) + 가이드라인** 형태입니다. 개원 시 원장님 검수 후 확정.
>
> 의료광고법 주의: "최고", "유일", "100% 효과", 환자 후기 직접 인용은 회피.
> "체계적", "맞춤형", "정성스러운", "전문성을 갖춘" 같은 안전 표현 위주.

---

## 0. 카피라이팅 공통 톤

- **2~3인칭 혼용**: "환자분께", "저희 백세한방병원은"
- **문장 길이**: 한국어 한 문장 35자 이내, 한 문단 3~4문장 이내
- **숫자 강조**: "1년 365일", "20년 임상 경험" 같은 수치는 항상 굵게 + accent 색
- **전문 용어 풀이**: 한방 용어는 첫 등장 시 괄호로 풀어 설명 (예: "추나(밀고 당기는 수기치료)")
- **종결어미**: 정중하게 "~합니다", "~드립니다" — 너무 딱딱한 "~함" 회피
- **금기 표현**: "치료를 보장합니다", "100% 완치", "최고의", "유일한", "기적의"

### 섹션 2 — Clinic Card Grid (6+1)

타이틀: **진료안내** (overline) / **부위와 증상에 따른 정성스러운 진료** (h2)

카드 6개 (진료) + 1개 (의료진). 데스크톱 4열 (1+1+1+1 / 1+1+1+1 — 첫 카드 가로 2배), 모바일 1열.

| 카드 | 제목 | 한 줄 설명 |
|---|---|---|
| 척추·관절 (큰 카드) | 척추·관절 클리닉 | 디스크부터 오십견까지, 통증의 원인을 함께 봅니다 |
| 추나·교정 | 추나·체형교정 | 틀어진 자세를 바로잡는 비수술 한방 교정 |
| 교통사고 | 교통사고 한방치료 | 교통사고 후유증, 자동차보험으로 진료받으세요 |
| 면역·만성 (큰 카드) | 면역·만성 클리닉 | 떨어진 면역과 누적된 피로, 한방으로 회복합니다 |
| 여성·소아 | 여성·소아 클리닉 | 산후·생리·갱년기·성장, 시기별 맞춤 진료 |
| 의료진 (특수) | 의료진 소개 | 한방 전문성을 갖춘 의료진이 진료합니다 |

각 카드 hover 시 이미지 1.05 scale + 화살표 아이콘 우측으로 슬라이드.

데스크톱 3열, 모바일 1열 스택.

**(a) 공지사항**
- 최근 5건 리스트, 제목 + 날짜
- "더보기 +" 링크 → `/community/notice`

**(b) 블로그 / 웹툰 / 영상**
- 네이버 블로그 최신 글 3건 또는 자체 매거진 카드 3개
- 외부 링크 처리

**(c) 자주찾는 서비스**
- 5개 아이콘 그리드: [서류발급] [비급여] [온라인상담] [한약안내] [블로그]
- 각 아이콘은 커스텀 SVG, hover 시 accent 색

### 섹션 5 — Location + Hours (지도 + 진료시간)

(전화번호는 모두 `tel:` 링크)

## 9~14. 진료 카테고리 페이지 공통 템플릿

각 카테고리(척추, 추나, 교통, 면역, 여성·소아)의 1depth 진입 페이지(`/spine`, `/posture` 등)와 2depth 상세 페이지는 **같은 템플릿**으로 빌드.

### 1depth 인트로 페이지 템플릿

1. **서브 비주얼** + 카테고리 타이틀
2. **이 카테고리에서 다루는 증상** — 4~6개 칩 그리드
3. **백세한방병원의 OOO 클리닉이 다른 점** — 3~4개 USP 카드
   - 예: 정확한 진단 / 부위별 맞춤 치료 / 한약·침·추나 통합 / 사후 관리
4. **세부 증상별 페이지 링크** — 카드 그리드 (각 2depth로 이동)
5. **치료 과정** — 단계별 인포그래픽 (4~5단계)
6. **자주 묻는 질문** — 아코디언 5개
7. **하단 CTA** — 전화 / 온라인 상담

### 2depth 상세 페이지 템플릿 (예: `/spine/disc`)

1. **서브 비주얼** + 증상명
2. **증상이란?** — 정의 + 일반적 원인 (의학적 사실 정도만)
3. **이런 분들께 추천드립니다** — 체크리스트 5~7개
4. **백세한방병원의 치료 방법** — 카드 4~6개
   - 한약 / 침 / 약침 / 봉침 / 추나 / 물리치료 / 운동재활 등
5. **치료 사례·과정 안내** (구체적 환자 사례 절대 X — 일반 단계만)
6. **자가 관리 / 생활 습관 가이드** — 리스트
7. **자주 묻는 질문** — 아코디언 3~5개
8. **CTA**

### 분량 가이드
- 1depth 페이지: 본문 800~1200자
- 2depth 페이지: 본문 1500~2500자

## 15. `/community/notice` — 공지사항

### 리스트 페이지
- 카테고리 필터 (전체 / 진료안내 / 휴진 / 이벤트)
- 검색
- 페이지네이션 (10개씩)
- 표 형태: [번호] [카테고리] [제목] [작성일] [조회수]

### 상세 페이지
- 타이틀 + 메타 (카테고리, 작성일, 조회수)
- 본문 (MDX 또는 리치 텍스트)
